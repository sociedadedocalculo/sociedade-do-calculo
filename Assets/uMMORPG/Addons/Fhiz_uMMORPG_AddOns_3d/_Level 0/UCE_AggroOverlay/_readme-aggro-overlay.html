AGGRO OVERLAY 2019.100

This Aggro Overlay Indicator allows monsters to show an icon above their head when switching to a new player target.

Core Changes

Many AddOns require changes to the uMMORPG3d core scripts in order to function. The most common core changes are listed in a extra ReadMe contained in the UCE Tools AddOn (see HERE). The changes for this AddOn are repeated below as well:

This AddOn requires the following core change(s):

[Click] AggroArea.cs (ca. line 15)

EXCHANGE this code block: 


// same as OnTriggerStay
    void OnTriggerEnter(Collider co)
    {
        Entity entity = co.GetComponentInParent<Entity>();
        if (entity) owner.OnAggro(entity);
    }
...COMPLETELY with the code block below (adds a new hook): 

// same as OnTriggerStay
    void OnTriggerEnter(Collider co)
    {
        Entity entity = co.GetComponentInParent<Entity>();
        if (entity) owner.OnAggro(entity);
        
        
        if (owner is Monster)
        	Utils.InvokeMany(typeof(Monster), (Entity)owner, "OnClientAggro_", entity);				// Fhiz Core Change
        
    }

INSTALLATION PROCESS

Integration Steps

  1. [Required] Don't forget the core change.
  2. [Required] Add the prefab to your monster prefab
  3. [Required] Link the prefab to your monster in the inspector (near text overlays)
  4. [Optional] Optionally Combine with my free "Floater" AddOn to make the Icon float.
